# Predictive Customer Churn Analysis in African Fintech

## Business Context

Customer retention is a critical challenge for African fintech companies operating in competitive and price-sensitive markets. Reducing churn directly impacts customer lifetime value and long-term profitability..

## Business Problem

How can predictive analytics identify customers at high risk of churn and support targeted retention strategies?

## Methodology

- Data Cleaning & Preparation (R, tidyverse)
- Exploratory Data Analysis
- Feature Engineering (Engagement Score, Early Risk Flag)
- Predictive Modeling:
  - Logistic Regression
  - Random Forest
- Model Evaluation (ROC Curve, AUC)

## Key Findings

- Churn is strongly concentrated among low-tenure customers.
- Low engagement significantly increases churn probability.
- Customers using automated payment methods show higher retention.
- Random Forest outperformed logistic regression in predictive performance.

## Strategic Recommendations

1. Strengthen onboarding during first 90 days.
2. Implement engagement scoring system.
3. Promote automated payment incentives.
4. Develop long-term subscription benefits.

## Model Performance

Random Forest achieved strong predictive performance (AUC > X), demonstrating reliable churn risk detection.
